Explore my art

I've recently became interested in CSS art. Creating these has greatly improved my knowledge of CSS. Plus, it's important to have a little fun!

CSS Art

CSS Cactus

Here I made a cactus image using CSS and SASS. This was done through a tutorial walkthrough so I could get a feel of how CSS artists format their procedures and design processes.

CSS Boba

Here I made a boba image as requested by a friend of mine. Using CSS and SASS.

This was done through my own facilitation. I wanted to create my own CSS art right away. Luckily a friend of mine suggested I make a boba art. And so I did!

CSS UFO

CSS UFO made using CSS and SASS. This also includes a bit of animation where the UFO is gliding and eventually disappears.

This was the second walkthrough I did. This took longer to make which was good practice when it came to learning the design processes of different CSS art projects. Afterwards, I created the Boba art image.
