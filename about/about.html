A B O U T M E

Nyrell Leonor

Since beginning my journey learning Front-End in 2021, I've created projects that I didn't even think of being able to create when I first started. From simple card designs using HTML and CSS to building multi-page websites alongside API's!


My learning journey has been a great eye-opener and something that I enjoy doing. Currently, I am strengthening my skills as a front-end developer by constantly creating projects, learning frameworks, and refining my knowledge in HTML, CSS, and JavaScript.

Learning and creating with...

HTML5

The markup language used for structuring and presenting content.

CSS3

The standard used in the styling and formatting of web pages.

SASS

A preprocessor language that is compiled into CSS. Sass lets you use features that do not exist in CSS, like variables, nested rules, mixins, imports, inheritance, built-in functions and much more!

JavaScript

A text-based programming language used both on the client-side and server-side that allows you to make web pages interactive.

Git

An open source software for distributed version control. It is used to track changes in the source code, enabling multiple developers to collaborate on non-linear development.

Tailwind

An open source CSS framework.

React.js

An open source JavaScript library for building user interfaces by creating UI components.

Node Package Manager

The world's largest software registry. It is a command line tool that installs, updates, or uninstalls Node.js packages for an application.

TypeScript

TypeScript is a strongly typed programming language that builds on JavaScript. It is designed for the development of large applications and transpiles to JavaScript.
